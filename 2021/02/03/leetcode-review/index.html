  27. BN 操作的实现

    def bn_forward(x, gamma, beta, eps):
    x_mean = np.mean(x, axis=(0, 2, 4), keepdims=true)
    x_var = np.var(x, axis=(0, 2, 4), keepdims=true)
    
    hat_x = (x - x_mean) / np.sqrt(x_var + eps)
    out_x = gamma * hat_x + beta
    
    return out_x
